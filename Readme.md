<h1>다락방 딥러닝 스터디 공룡게임 AI 프로젝트</h1>
getTrexPlayData.py -> 데이터셋을 얻는 과정
<br>
play.py -> 데이터셋을 기반으로 러닝을 진행
<br>
test.py -> 학습한 모델을 기반으로 게임을 플레이함.
